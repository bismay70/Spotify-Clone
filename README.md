🎵 Spotify Clone

A minimal Spotify-inspired music player built to explore UI design and audio playback features.
-----
🚀 Features

▶️ Play / Pause music
⏭️ Next / Previous track
📃 Playlist view
⏱️ Progress bar & duration
🔊 Volume control 
-----
🛠️ Tech Stack
Frontend: React, TailwindcSS
Backend: Express,MongoDB
